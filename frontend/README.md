# He Walks Unseen (Web)

This folder contains the Vite + React + TypeScript web app.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Test

```bash
npm test
```

## Lint / Format

```bash
npm run lint
npm run format
```

## Assets

- Levels and themes should live in `frontend/public/data/`.
- Core logic lives in `frontend/src/core/` and must remain UI-agnostic.
